Alternative(s) to VirtualBox

There are other alternatives for hosting VMs on your own computer, e.g.VMware. TheVMware workstation playeris free for download, however, it only runs on Windows and Linux.

Running 64-bit virtual machines

If you want to create your own VM and install a 64-bit OS, you will need to enable the feature in the BIOS, namely VT-x for Intel processors, or AMD-v for AMD processors

  • Enable VT-x or AMD-v in BIOS setting
  • Checklist on settings (if enabling VT-x or AMD-v along does not work)
